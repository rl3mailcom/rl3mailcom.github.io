While there are many resources available for you as you build one thing should be determined before your start. Are you building a website or a web application (webapp). Knowing this will determine which set of resources you will need. A website would lend to more free or nominal fee resources for testing, hosting, and analytics.

Most of the web we don't see. It exists in what we call the 'dark web'. It will be interesting to see how Web 3.0 handles the dark web.
