WEB 101

I used to wonder why they called it web. Why is it the World Wide Web (www)? Well, I think it is because when you put it all together it looks like a web! Web Development, Full-Stack, Front-End vs. Back-End Development, and Internet Resources are how the web comes together. These groups are broken down in this way to introduce some aspects of how the web continues to evolve. Web Development looks at how to get started. Front-End introduces the forward-facing pieces of web site/application development. Back-end deals with the foundational, unseen or behind the scenes part of web site/application development. Internet Resources is the place where I cover some topics that relate to the Internet and to web development and does not fit into any of the other categories.

I created an image file like a site map relating these topics and the sub-topics. As I was connecting them is when I began to wonder and when it came to me about the ‘web’. The evolution of the world-wide-web is interesting. All the tools and utilities that have been created as part of the web evolution are amazing.

Learning about the world-wide-web as a beginner to being a first level developer to becoming an advanced developer is a great endeavor. There are more tools than I could have ever imagined. Hopefully the information presented within this site will convince someone to become a web developer or ‘level-up’ their current skill level.
